工作職掌

1. 系上所有 Domain Name Server 設定、管理、備份

(正解、反解)

2. 系上有線網路/無線網路維護管理

有線網路：Public IP分配、DHCP IP分配、NAT、Loop排除
無線網路：無線訊號品質監測、測速、AP干擾

3. 系上有線網路架構規劃/修改

(Gateway接管、VLAN切分)

4. 系上無線網路架構規劃/修改

(系館無線網路順暢、802.1x帳號驗證登入)

5. 主機房網管設備建置

(主機代管區Switch建置、各機櫃網路線整線)

6. 系上網路流量監控平台

(監控各樓層、各辦公室、教室、實驗室流量)

7. 協助系上教職員、學生任何網管事件

(網路不通、網路通但無法遠端、區網通無法對外上網等等…)

8. 協助計中處理資安事件

(通知使用者中毒，並填寫資安事件單存查)

9. 系上網路資訊設備評選、採購

(中信局：評選有技術能力的配合廠商購買
非中信局：評選有技術能力外，還需要價格合理)

10. 網管資產管理

(確認每項資產用途、位置、使用年限，超過使用年限需注意是否更換新品或是與廠商簽訂維護合約)
