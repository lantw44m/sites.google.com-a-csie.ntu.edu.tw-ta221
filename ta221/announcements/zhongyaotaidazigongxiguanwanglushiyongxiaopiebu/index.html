[重要] 台大資工系館網路使用小撇步

為了提升大家使用網路的感受，系上網管團隊整理了一些重要的小知識：

# VPN ---------------------------------------------------------------
VPN 可以讓你假他人 IP 在網路世界徜徉，有些網站或資源只願意讓特定的 IP 碰，IP 的世界是不平等的。台大的 IP 有如資產階級，在網路世界無往不利，透過台大的 VPN 可以讓你享受特權 (下載台大付過錢的論文、電子書、資料等，但請不要透過程式大量下載，系上幾乎每學期都有同學被抓)。而台大資工系的 IP 除了享有台大 IP 的好處外，更可以無縫接軌系館有線網路架構，與系館內部的電腦溝通有無、互利共生。想要台大資工系 IP 的特權就使用台大資工系自己的 VPN 吧！

使用說明：https://esystem.csie.ntu.edu.tw/nalab/vpn

# WiFi ----------------------------------------------------------------
無線網路頻段可分為2.4GHz以及5GHz兩種頻段，兩者特性如下：
- 2.4GHz 的訊號覆蓋範圍較大，但網速較慢
- 5GHz 的網速快，但相對覆蓋範圍較小
德田館中由系上網管維護的無線網路主要有csie及csie-5G，其中csie支援2.4GHz及5GHz頻段（由設備決定較適合的頻段），csie-5G則是只支援5GHz頻段。

＊要在德田館享受最舒適的無線網路體驗，建議優先連接csie，而只在訊號穩定且有較高網速需求時使用csie-5G。至於計中提供的NTU及ntu-peap則是建議在系上網路無法使用時再考慮使用。
＊請課程助教詢問外系修課同學是否有使用系上無線網路的需求，若有請課程助教至https://esystem.csie.ntu.edu.tw/ 申請帳號。
＊此外，若是在系館中遇到無線網路使用上的問題，請填寫問題回報（https://goo.gl/forms/FIPwhKpgvgaxU64K3） 以利網管團隊改善狀況。
＊若有其他需求或疑問，請來信至 ta221@csie.ntu.edu.tw 。

# Wired --------------------------------------------------------------
若同學於系上有使用有線網路的需求，裏新館內的座位皆附有網路孔，同學可自備網路線至裏新館使用。接上網路線後可直接透過DHCP取得IP位址並連上網際網路。

# Security -----------------------------------------------------------
你知道瀏覽器「綠色鎖頭」所代表的安全意義嗎？
看似，你和「這個網址」之間建立了「安全」的連線，然而......

＊你確定「這個網址」打對了嗎?
釣魚網站可能會用相近的錯字，例如將 apple.com 改成 "app1e"，或是使用子域名（subdomain）騙取信任（延伸閱讀：另一種假網址）

＊這連線真的「安全」嗎？ 
https 連線，意指你到上述網址的連線不會遭到第三方竊聽，不是代表正在進行的操作是安全的，即使進到釣魚網站還是可以顯示綠色鎖頭，而且成本非常低廉。 （初級延伸閱讀：網址的價錢、免費的 https 證書；駭客級延伸閱讀： DNS 快取汙染 、驗證 DNS 紀錄）

# For Lab Users ---------------------------------------------------
各實驗室除了擁有一些 Public IP（140.112.x.x）外，同時也擁有一個 Static Private IP 的網段（172.16.x.x），這個網段內的 Private IP 可供實驗室自由使用，且外部網路無法連線至此網段，僅有系館內的設備得以連線至該 Private IP。應用的例子如：實驗室內的印表機、機房內的設備、以及各種需要被保護不能被外界主動接觸到的機器，都可以利用這個網段。這些網段在系館中屬於 vlan 7，預設的孔位就是各實驗室中 HP Switch 的第 7 個孔位，只要接上去即可使用。若在系館外仍有連線到此網段內機器的需求，可利用 CSIE VPN 或工作站 SSH Tunnel 進行連線。

＊若各實驗室不知道自己被分配到的 Static Private IP 網段為何，請洽 R217 或寄信至 ta221@csie.ntu.edu.tw。
＊若實驗室 Public IP 無法使用，請至計中（http://cert.ntu.edu.tw/Module/Index/ip.php）查詢IP是否被封鎖。
