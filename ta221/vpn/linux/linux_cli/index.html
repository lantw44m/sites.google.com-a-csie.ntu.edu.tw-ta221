使用 OpenVPN 指令在終端機連線

如何在 Linux 用 OpenVPN 指令在終端機連上 CSIE VPN

以下內容在 Ubuntu 16.04 上測試通過。

1. 安裝 OpenVPN 套件。

發行版　套件名稱　安裝方法
Archopenvpnpacman -S
CentOSopenvpnyum install（必須先啟用 EPEL 套件庫）
Debian / Ubuntuopenvpnapt install
Fedoraopenvpndnf install
Gentoonet-misc/openvpnemerge
Mageiaopenvpnurpmi
openSUSEopenvpnzypper install

2. 到這裡下載 VPN 設定檔：config.ovpn。

指令：wget https://sites.google.com/a/csie.ntu.edu.tw/ta221/download/config.ovpn。 

3. 輸入 openvpn 指令和工作站的帳號密碼，這樣就成功連上 CSIE VPN 囉！

指令：sudo openvpn config.ovpn 

4. 執行 ip route 後，出現了若干條 172 開頭的路由規則，這也是成功的象徵喔！

5. 若要結束 VPN 連線，在執行 openvpn 指令的終端機按下 Ctrl + C 即可。

如果有任何問題，請寄信至 ta221。
