網路檢測

網路不通時，簡易自我檢測方法（Windows）
  1. 請點選「開始」→「執行」→輸入「cmd」，會彈出命令提示字元視窗 。

  2. 請打「ipconfig」，檢查 IP 是否設定正確，或有無抓取到 IP。

  3. 請先檢測使用的 gateway 連線是否正常, ping 140.112.x.254

  4. 請 ping 系上 DNS: 140.112.30.21 或 計中 DNS: 140.112.254.4
    如果有通代表正常，不通請使用下列方式

  5. 請 tracert 140.112.30.21  或 計中 DNS: 140.112.254.4
    查看網路斷在哪個路由器，並把該畫面或文字截取給網管室。

  6. 臺大計中網路測速
