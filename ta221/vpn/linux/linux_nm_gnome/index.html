使用 NetworkManager 連線：GNOME 3

如何在 Linux 用 NetworkManager 連上 CSIE VPN (GNOME 3)

以下內容在 Arch Linux 2016-11-19 上測試通過。

1. 安裝 NetworkManager 的 OpenVPN 外掛程式。

發行版　套件名稱　安裝方法
Archnetworkmanager-openvpnpacman -S
CentOSNetworkManager-openvpn-gnomeyum install（必須先啟用 EPEL 套件庫）
Debian / Ubuntunetwork-manager-openvpn-gnomeapt install
FedoraNetworkManager-openvpn-gnomednf install
Gentoonet-misc/networkmanager-openvpnemerge
Mageianetworkmanager-openvpnurpmi
openSUSENetworkManager-openvpn-gnomezypper install

2. 重新啟動 NetworkManager 以載入剛才安裝的外掛程式。

如果您不知道如何操作，請參考發行版的說明文件，或是試試看 sudo systemctl restart NetworkManager。

3. 到這裡下載 VPN 設定檔：config.ovpn。

4. 開啟「設定值」。

5. 選取「網路」。

6. 按左下方的「＋」。

7. 按「從檔案匯入…」，選擇剛才下載的 config.ovpn 檔案。

8. 輸入工作站帳號密碼，如果不想儲存密碼，可以選擇在每次連線時才輸入密碼。

注意 Type 必須選擇「密碼」，「CA 憑證」欄位必須有選取檔案後才能按下「加入」。 如果您的系統版本較舊，沒有自動填入「CA 憑證」，請到這裡下載憑證檔案：ca.crt，再手動指定檔案路徑。 

9. 如果不想使用 CSIE VPN 作為預設對外的網路連線，請至 IPv4 和 IPv6 分頁勾選「只在使用這個連線的網路資源時才使用此連線」。

10. 設定完成後選取視窗左側的 CSIE VPN，按下右上方的開關圖示即可連線。

11. 連線成功！
